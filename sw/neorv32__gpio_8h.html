void neorv32_gpio_pin_change_config ( uint32_t  pin_sel)

Configure pin-change IRQ mask for input pins.

Note
The pin-change IRQ will trigger on any transition (rising and falling edge) for any enabled input pin.
Parameters
[in]pin_selMask to select which input pins can cause a pin-change IRQ (32-bit), 1 = pin enabled.
