Configure NCO channel. The NCO control register bits are listed in NEORV32_NCO_CT_enum.

Parameters
[in]channelChannel number (0,1,2).
[in]modeOperation mode: 0=normal (50% duty cycle), 1=pulse-mode.
[in]idle_polIdle polarity (0 or 1).
[in]oeEnable output to processor top pin when set.
[in]prscClock select / clock prescaler, see NEORV32_CLOCK_PRSC_enum.
[in]pulseSelect pulse length (in clock-prescaler cycles) for pulse-mode. See data sheet.
